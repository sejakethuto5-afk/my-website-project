## Project Overview

This project is a multi-page website for a fictional bakery called Sweet Crumbs.
The website provides information about the bakery, its products, and allows users to make enquiries.

## Website Pages

* Home Page (index.html)
* About Page (about.html)
* Products Page (products.html)
* Contact Page (contact.html)
* Enquiry Page (enquiry.html)

## Features

* Navigation menu linking all pages together
* Images and text content about the bakery
* Multiple HTML pages connected with links

## Goals and Objectives

* To create a functional multi-page website
* To practice using HTML structure and tags
* To ensure all navigation links work correctly
